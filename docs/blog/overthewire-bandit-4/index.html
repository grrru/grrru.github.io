Level 32

After all this git stuff, it’s time for another escape. Good luck!

git 문제는 끝났다!

1ssh bandit32@bandit.labs.overthewire.org -p 2220
2
3WELCOME TO THE UPPERCASE SHELL
4>>
5>>

접속하면 UPPERCASE SHELL라는 쉘이 실행된다. 입력하는 모든 command를 대문자로 치환하는 쉘이다.

1>> $SHELL
2WELCOME TO THE UPPERCASE SHELL

문제 해결 후 회상

$0을 입력하면 현재 실행 중인 쉘의 실행 경로가 입력되어 쉘이 재실행된다.
이 문제의 세팅을 보면 login shell은 uppershell이다.

1cat /etc/passwd | grep bandit32
2bandit32:x:11032:11032:bandit level 32:/home/bandit32:/home/bandit32/uppershell
3
4echo $SHELL
5/home/bandit32/uppershell
1$ $0
2$ echo $0
3sh

$0을 실행하면 sh가 실행되는데, 이는 이 명령을 실행하던 shell이 sh였다는 뜻이다.
로그인 직후 uppershell가 실행되고 얘가 sh를 실행했다는 뜻(?)인 듯 하다.
여기서 /usr/bin/bash를 실행하면 bandit에서 보던 bash shell이 실행된다.

1$ /usr/bin/bash
2bandit33@bandit:~$

해결 방법

1$0
2cat /etc/bandit_pass/bandit33
